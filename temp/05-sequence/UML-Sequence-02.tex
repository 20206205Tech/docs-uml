\paragraph{Sơ đồ tuần tự Luồng Nghe thử mẫu giọng nói của nhân vật}

Luồng này mô tả quá trình Guest thực hiện click nghe thử giọng nói của trợ lý ảo. Dữ liệu âm thanh sẽ được stream trực tiếp từ Storage Bucket (Cloudflare R2 / Supabase Storage) về Web Frontend.

\subparagraph{Các thành phần tham gia}

\begin{table}[H]
\centering
\renewcommand{\arraystretch}{1.3}
\begin{tabular}{|l|l|p{5cm}|}
\hline
\textbf{Thành phần} & \textbf{Phân loại} & \textbf{Vai trò trong hệ thống} \\ \hline

Web Frontend & Component & Nhận và phát luồng âm thanh nghe thử. \\ \hline

\subparagraph{Mô tả tổng quan}

\begin{itemize}
\item Guest nhấn nút "Nghe thử" của một nhân vật bất kỳ trên trang chủ.
\item Web Frontend gửi yêu cầu tải/stream file âm thanh trực tiếp đến địa chỉ URL của Cloudflare R2 hoặc Supabase Storage mà không cần qua API Gateway hay Backend.
\item Cloudflare R2 / Supabase Storage trả về luồng dữ liệu âm thanh định dạng \texttt{.mp3} hoặc \texttt{.wav}.
\item Web Frontend nhận dữ liệu và thực hiện phát âm thanh cho Guest nghe.
\end{itemize}
