\paragraph{Sơ đồ tuần tự Luồng Theo dõi báo cáo VBPL Pipeline}

\subparagraph{Mô tả chi tiết}
\begin{itemize}
\item \textbf{Yêu cầu xem báo cáo:} Admin truy cập vào Dashboard quản trị. Web Frontend gửi hai yêu cầu qua Kong Gateway tới VBPL Pipeline Service:
    \begin{itemize}
    \item \texttt{GET /data-pipeline-vbplnew-service/documents/total}: Lấy tổng số văn bản luật đã thu thập và mốc thời gian cập nhật gần nhất.
    \item \texttt{GET /data-pipeline-vbplnew-service/workflows/summary}: Lấy số lượng văn bản đang nằm ở các bước xử lý (như cào dữ liệu, trích xuất text, chia nhỏ chunk, tạo nhúng vector RAG).
    \end{itemize}
\item \textbf{Xử lý thống kê:} VBPL Pipeline Service truy vấn các bảng thống kê và đếm số lượng bản ghi theo từng trạng thái bước trong Pipeline Database, sau đó phản hồi số liệu tổng hợp về cho Web Frontend.
\item \textbf{Biểu diễn thông tin:} Web Frontend nhận dữ liệu phản hồi, tổng hợp và vẽ biểu đồ trực quan (biểu đồ cột hoặc hình tròn) để Admin theo dõi trực tiếp sức khỏe của các workflow.
\end{itemize}
