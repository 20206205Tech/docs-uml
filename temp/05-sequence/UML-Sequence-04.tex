\paragraph{Sơ đồ tuần tự Luồng Đăng ký tài khoản bằng Email}

Luồng nghiệp vụ này mô tả chi tiết quá trình đăng ký tài khoản mới sử dụng địa chỉ Email và mật khẩu. Toàn bộ các yêu cầu đăng ký từ Web Frontend đều đi qua API Gateway (Kong) trước khi gửi tới Auth Service để tạo tài khoản ở trạng thái chờ xác thực.

\subparagraph{Mô tả tổng quan}

\begin{itemize}
\item Guest nhập địa chỉ Email và mật khẩu, nhấn nút Đăng ký trên Web Frontend.
\item Web Frontend gửi yêu cầu đăng ký qua API Gateway (Kong). Gateway thực hiện định tuyến và chuyển tiếp yêu cầu đến Auth Service.
\item Auth Service khởi tạo tài khoản mới trong Database ở trạng thái chưa kích hoạt (\texttt{email\_confirmed\_at = null}).
\item Auth Service yêu cầu Supabase Email gửi một thư điện tử chứa liên kết xác thực tài khoản (chứa token kích hoạt trỏ qua API Gateway Kong) tới hòm thư của Guest.
\item Auth Service phản hồi đăng ký thành công về cho Web Frontend thông qua API Gateway, Frontend hiển thị thông báo yêu cầu người dùng kiểm tra hòm thư điện tử.
\end{itemize}
