\subparagraph{Mô tả chi tiết}
\begin{itemize}
\item \textbf{Yêu cầu xem báo cáo:} Admin truy cập vào Dashboard quản trị. Web Frontend gửi yêu cầu lấy số liệu thống kê Pháp Điển bằng phương thức \texttt{GET /data-pipeline-phapdien-service/phapdien/summary} qua Kong API Gateway.
\item \textbf{Kiểm tra và Định tuyến:} Kong Gateway xác thực Bearer Token của Admin, sau đó chuyển tiếp yêu cầu tới Pháp Điển Pipeline Service.
\item \textbf{Thống kê dữ liệu:} Pháp Điển Pipeline Service thực hiện truy vấn cơ sở dữ liệu (Pipeline DB) để lấy tổng số chủ đề (\texttt{total\_chu\_de}), tổng số đề mục (\texttt{total\_de\_muc}), và tổng số nút thư mục (\texttt{total\_tree\_items}) thuộc bộ chỉ mục Pháp Điển. Dịch vụ tổng hợp các số liệu này và phản hồi về cho Web Frontend.
\item \textbf{Biểu diễn thông tin:} Web Frontend nhận dữ liệu phản hồi, tính toán tiến độ hoàn thành và vẽ biểu đồ trực quan giúp Admin dễ dàng theo dõi.
\end{itemize}
