\paragraph{Sơ đồ tuần tự Luồng Quản lý Engine giọng nói}

Luồng nghiệp vụ này mô tả quy trình Quản trị viên (Admin) tạo mới hoặc cập nhật Engine giọng nói trong Persona Service. Theo code hiện tại, Engine là cấu hình nội bộ gồm \texttt{code}, \texttt{name} và \texttt{is\_active}; bước đồng bộ dữ liệu ElevenLabs được mô tả riêng ở luồng đồng bộ Voices.

\subparagraph{Mô tả chi tiết}
\begin{itemize}
\item \textbf{Nhập cấu hình:} Admin mở trang Engine trong Admin Panel và nhập mã Engine, tên hiển thị cùng trạng thái hoạt động.
\item \textbf{Gửi yêu cầu lưu:} Web Frontend gửi yêu cầu \texttt{POST /code-persona-service/engines} hoặc yêu cầu cập nhật \texttt{PUT /code-persona-service/engines/\{engine\_id\}} qua Kong API Gateway kèm Bearer Token.
\item \textbf{Lưu cấu hình:} Persona Service thực hiện tạo mới hoặc cập nhật bản ghi Engine trong Persona Database.
\item \textbf{Hoàn tất:} Hệ thống phản hồi cấu hình đã lưu về Frontend để Admin nhìn thấy trạng thái cập nhật thành công.
\end{itemize}
