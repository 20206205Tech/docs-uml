\paragraph{Sơ đồ tuần tự Luồng Quản lý TTS Engine ElevenLabs}

Luồng nghiệp vụ này mô tả quy trình Quản trị viên (Admin) thiết lập hoặc cập nhật TTS Engine dùng nhà cung cấp ElevenLabs, bao gồm kiểm tra khóa kết nối, chọn model mặc định và lưu cấu hình để các giọng nói (Voices) có thể sử dụng thống nhất trong hệ thống.

\subparagraph{Mô tả chi tiết}
\begin{itemize}
\item \textbf{Nhập cấu hình:} Admin mở trang TTS Engine trong Admin Panel và điền các thông tin như mã engine, nhà cung cấp, tham chiếu khóa API, model mặc định, giọng nói mặc định và trạng thái hoạt động.
\item \textbf{Gửi yêu cầu lưu:} Web Frontend gửi yêu cầu \texttt{POST /persona-service/v1/tts-engines} hoặc yêu cầu cập nhật tương ứng qua Kong API Gateway kèm Bearer Token.
\item \textbf{Kiểm tra kết nối ElevenLabs:} Persona Service dùng khóa cấu hình để gọi \texttt{GET /v1/models} tới ElevenLabs API nhằm xác nhận cấu hình hợp lệ và lấy danh sách model khả dụng.
\item \textbf{Lưu cấu hình:} Persona Service thực hiện \texttt{UPSERT} bản ghi TTS Engine trong Persona Database, lưu trạng thái kiểm tra kết nối và thời điểm kiểm tra gần nhất.
\item \textbf{Hoàn tất:} Hệ thống phản hồi cấu hình đã lưu về Frontend để Admin nhìn thấy trạng thái kết nối ElevenLabs hợp lệ.
\end{itemize}
