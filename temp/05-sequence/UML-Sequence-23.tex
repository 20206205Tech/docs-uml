\paragraph{Sơ đồ tuần tự Luồng Thiết lập và đồng bộ giọng nói từ ElevenLabs}

Luồng nghiệp vụ này mô tả cách thức Quản trị viên (Admin) kích hoạt quá trình đồng bộ các mẫu giọng nói (Voices) từ nền tảng ElevenLabs về cơ sở dữ liệu của Persona Service để sử dụng cho tính năng Voice Agent.

\subparagraph{Mô tả chi tiết}
\begin{itemize}
\item \textbf{Yêu cầu đồng bộ:} Admin chọn chức năng đồng bộ trên giao diện. Frontend gửi yêu cầu \texttt{POST /voices/sync-elevenlabs} qua Kong tới Persona Service.
\item \textbf{Gọi API ElevenLabs:} Persona Service gửi yêu cầu \texttt{GET /v1/voices} tới ElevenLabs API kèm theo khóa xác thực API Key của hệ thống được lưu trong biến môi trường.
\item \textbf{Xử lý và Lưu trữ:} ElevenLabs trả về danh sách giọng nói dưới dạng JSON. Persona Service duyệt qua từng giọng nói, kiểm tra sự tồn tại trong Persona Database dựa trên Voice ID:
    \begin{itemize}
    \item Nếu chưa tồn tại, chèn mới bản ghi với thông tin tên, category, preview\_url và gán engine\_code là \texttt{elevenlabs}.
    \item Nếu đã tồn tại, thực hiện cập nhật lại thông tin mới nhất.
    \end{itemize}
\item \textbf{Hoàn tất:} Persona Service phản hồi mã trạng thái HTTP 200 OK trả về số lượng giọng nói đã đồng bộ thành công để hiển thị thông báo lên màn hình Admin.
\end{itemize}
